### 2.2.2 Inventory Data

This page use to show data accepted by system from page inventory data.

![](extracted_screenshot_46.png)

Figure Inventory Data Upload Result

**Inventory Data Result Page**

This page displays a historical log of the opening balance inventory records that have been successfully processed and loaded into the system. It aggregates various inventory sources into a single view for auditing and verification.

**Section 1: Data Filtering & Export**

- **Search & Filters**: Users can query the dataset using **FA Code**, **Source Types** (supporting V1, DC, and BV1), and specific **Week/Year** parameters.
- **Refresh**: Synchronizes the table with the most recent database entries.
- **Export Action**: An orange button to download the current view or filtered dataset for further reporting.

**Section 2: Inventory Data Table**

The grid provides a detailed breakdown of stock levels across different product and location segments:

| **Column Name** | **Description** |
| --- | --- |
| ID | Unique sequence identifier for the record entry. |
| Source Type | The origin of the inventory data (e.g., INVENTORY\_V1, INVENTORY\_DC, INVENTORY\_BV1). |
| Week / Year | The reporting period (e.g., Week 48 of 2025). |
| FA Code | The Finished Article identification code. |
| Brand Code | The code identifying the specific product brand (e.g., DSS16, MBR20). |
| FA Type | Product sub-category (e.g., SKM, SKT). |
| Long Speaking Code | The full descriptive name of the article. |
| Plant | The specific code for the plant or distribution center (e.g., ID24, ZD4A). |
| Location Name | Human-readable location description (e.g., Plant Surabaya, DC Cikarang). |
| Stock Box | Current inventory level measured in boxes. |
| Stock Stick | Current inventory level measured in individual units/sticks. |
| Stock Custom | Specific inventory counts for custom units (used primarily by BV1 sources). |
| UOM Custom | The Unit of Measure for the custom stock (e.g., CARTON). |
| Uploaded By | The username of the person who uploaded the data batch. |

**Section 3: Navigation & Sample Info**

- **Entries Count**: A footer indicating the number of records being shown (e.g., "Showing 5 of 5 entries").
- **Pagination**: Standard "Prev/Next" and page number controls for navigating through large datasets.
